# Changelog

Все изменения проекта Liquid Torrent документируются в этом файле.

## [1.3.0] — 2026-05-17

### 🚀 Оптимизация производительности

- **Кэш метаданных** — файлы и трекеры торрента кэшируются один раз, не пересоздаются каждый тик
- **Лёгкий polling** — основной цикл UI запрашивает только лёгкие данные (без файлов/трекеров), снижение IPC нагрузки с ~2 МБ до ~5 КБ на тик
- **Ленивая загрузка RightPanel** — полная информация (файлы, трекеры) загружается только при выборе торрента
- **React.memo на TorrentCard** — карточки не перерисовываются если progress/speed/state не изменились
- **Мемоизация Sidebar** — счётчики фильтров обёрнуты в `useMemo`
- **Параллельное восстановление** — торренты при запуске загружаются одновременно (`Promise.allSettled`)

### 🐛 Исправления

- Исправлена захардкоженная версия в AboutModal (теперь берётся из `__APP_VERSION__`)
- Удалены неиспользуемые импорты (`nativeTheme`, `Download`, `Upload`, `CheckSquare`)
- Очищен кэш при удалении торрента (`staticCache.delete`)

### 📦 Инфраструктура

- Версия обновлена до 1.3.0
- Портативный EXE: `LiquidTorrent-1.3.0-portable.exe`

## [1.2.0] — 2026-05-16

### ✨ Первый релиз

- Полноценный торрент-клиент на Electron + React + WebTorrent
- Тёмная тема с неоновыми акцентами и glassmorphism
- Поддержка .torrent файлов и магнет-ссылок
- Системный трей с контекстным меню
- Дерево файлов торрента
- График скорости в реальном времени
- Индивидуальные лимиты скорости для торрентов
- Ассоциация с файлами `.torrent` и протоколом `magnet:`
- Настройки: лимиты скорости, авто-стоп раздачи, автозапуск
- Портативный EXE без установки
