# Changelog

Все изменения проекта Liquid Torrent документируются в этом файле.

## [1.3.0] — 2026-05-17

### 🏗️ Архитектура

- **WebTorrent вынесен в `utilityProcess`** — весь тяжёлый код (SHA1 хеширование, TCP подключения, piece verification) теперь работает в **отдельном OS-процессе**. Main process Electron стал лёгким прокси (~15KB), UI больше не зависает.

### 🚀 Оптимизация производительности

- **`maxConnections: 55`** (по документации WebTorrent) — было 200, вызывало 850+ TCP соединений и DDoS самого себя
- **`skipVerify: true`** при восстановлении торрентов — не хешировать 155ГБ при каждом запуске
- **`opts.paused`** — торренты добавляются в paused-состоянии через API, без лишних connect/disconnect циклов
- **`storeCacheSlots: 20`** — ограничение кэша хранилища для экономии RAM
- **Debounced save** — состояние пишется на диск асинхронно с 2с задержкой, не блокируя event loop
- **Трекеры сокращены** с 10 до 4 самых надёжных — меньше лишних соединений
- **Кэш метаданных** — файлы/трекеры кэшируются один раз, не пересоздаются каждый тик
- **Лёгкий polling** — UI запрашивает только лёгкие данные (без файлов/трекеров)
- **React.memo на TorrentCard** — карточки не перерисовываются если данные не изменились
- **Мемоизация Sidebar** — счётчики фильтров обёрнуты в `useMemo`
- **Параллельное восстановление** — торренты при запуске загружаются одновременно

### 🐛 Исправления

- **Критический баг:** `buffer.toString('hex')` — конвертация 5-10МБ `.torrent` файла в hex-строку при проверке дубликатов. Заменено на `buffer.includes(hashBuf)`
- Исправлена захардкоженная версия в AboutModal (теперь `__APP_VERSION__`)
- Удалены неиспользуемые импорты (`nativeTheme`, `Download`, `Upload`, `CheckSquare`)

### 📦 Инфраструктура

- electron-vite multi-entry build (index + torrent-worker)
- Портативный EXE: `LiquidTorrent-1.3.0-portable.exe`

## [1.2.0] — 2026-05-16

### ✨ Первый релиз

- Полноценный торрент-клиент на Electron + React + WebTorrent
- Тёмная тема с неоновыми акцентами и glassmorphism
- Поддержка .torrent файлов и магнет-ссылок
- Системный трей с контекстным меню
- Дерево файлов торрента
- График скорости в реальном времени
- Индивидуальные лимиты скорости для торрентов
- Ассоциация с файлами `.torrent` и протоколом `magnet:`
- Настройки: лимиты скорости, авто-стоп раздачи, автозапуск
- Портативный EXE без установки
